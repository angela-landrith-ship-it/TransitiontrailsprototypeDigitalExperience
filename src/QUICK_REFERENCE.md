```

2. Click "Assessments" tab
3. Choose assessment type:
   - Skills IQ (monthly)
   - Self-Assessment (quarterly)

```

```
1. Go to Community
2. Click "Group Sessions" tab
3. Browse available sessions
4. Click "RSVP" on desired session
5. Add to calendar
6. Join at scheduled time
```

#### Get Help from Penny
```
1. Click Penny button (bottom-right) or press /
2. Type or speak your question
3. Review Penny's response
4. Click suggested actions if needed
5. Ask follow-up questions
```

## ğŸ’¡ Pro Tips

### Maximize Your Experience

1. **Use Penny's Focus List as Your Daily Guide**
   - Don't worry about remembering everything
   - Penny prioritizes for you based on AI analysis
   - Trust the Focus List - it knows your deadlines and dependencies

2. **Engage in Community Daily**
   - Even just one message per day
   - Builds connections and earns points
   - Cohort support is invaluable

3. **Check in with Coach Regularly**
   - Don't wait for scheduled 1:1s
   - Quick Slack questions welcome
   - Proactive communication = better support

4. **Test Your Code Before Submitting**
   - Use Testing Dashboard in Capstone
   - Run unit tests locally
   - Get peer review before requesting coach review

5. **Use LinkedIn Profile Coach**
   - Keep profile updated as you learn
   - Share milestones (earn visibility)
   - Builds your professional brand

6. **Take Assessments Seriously**
   - Honest self-assessment = better coaching
   - Track growth over time
   - Identify areas for focus
